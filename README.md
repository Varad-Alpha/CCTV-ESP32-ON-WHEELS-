# CCTV-ESP32-ON-WHEELS-

Another project of mine because an CCTV on wheels is the thing that fits in perfectly with the jigsaw puzzle of humanity,

How it works(it's a habit now, deal with it):ESP32-CAM SURVEILLANCE THINGY is an amazing supercalifragilisticexpalidocious robot that streams live video using its state of the art standardized ESP32CAM MODULE. The robot also has a web server that provides access to a control interface. The human can connect to the robot’s Wi-Fi network. The human can then use cute buttons provided in a web page to control the movement of the robot. The ESP32 chip processes HTTP requests for movement. The video is streamed in real time to the browser(yayyy live stream).
